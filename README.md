# Rolled 🚀

**Rolled** is an all-in-one SaaS platform designed for small-to-medium businesses to seamlessly manage invoicing, payroll, and employee directories from a single dashboard. Built with a premium, modern aesthetic using **Next.js 14**, **TailwindCSS**, and **Firebase**.

## ✨ Core Features
- **Smart Invoicing**: Create branded, dynamic invoices with automated tax calculations and one-click email delivery (via Resend).
- **Payroll & Salary Slips**: Run monthly payrolls, calculate income tax based on custom tax slabs, and instantly distribute digital salary slips to your staff.
- **Employee Directory**: Manage employee profiles, departments, active statuses, and salary structures.
- **Financial Dashboard**: Get a high-level view of your business health with beautiful charts tracking revenue collections vs. payroll expenses.
- **Role-Based Access**: Secure, centralized authentication allowing distinct access levels for Admins, HR Managers, Accountants, and Employees.

## 🛠️ Tech Stack
- **Frontend**: Next.js 14 (App Router), React, Tailwind CSS, Recharts, Lucide Icons
- **Backend**: Next.js API Routes, Firebase Admin SDK, Cloud Firestore
- **Integrations**: Resend API (Email Automation)
- **Deployment**: Highly optimized and 100% Vercel-ready

## 🚀 Quick Start
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `npm install`
3. Copy the environment template: `cp .env.example .env.local`
4. Add your Firebase and Resend credentials to `.env.local`.
5. Start the development server: `npm run dev`
6. Open `http://localhost:3000` in your browser.
